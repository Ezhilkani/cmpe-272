#All the JSON format for DB creation for Assignment 2

## Books Json
Add details of the book
{
    "books": [{
            "isbn": "string",
            "title": "string",
            "subtitle": "string",
            "author": [""],
            "edition": "",
            "published": "date-time",
            "publisher": "string",
            "pages": <int>,
            "price": <int>,
            "description": "string",
            "website": "url"
        }
    ]
}

## Book inventory
- Keep track of the number of books available.
- use this to keep track of the book that has been ordered

{
    "book_inventory": [{
            "isbn": "string",
            "quantity": <int>,
        }
    ]
}
